# homebrew-tap

Homebrew formulae that allows installation of Tracer tools through the Homebrew package manager.

## Installation

```bash
brew tap Tracer-Cloud/tap
brew install <FORMULA>
```

## Formulae

| Repository | Formula                         | Description                                                                        |
| ---------- | ------------------------------- | ---------------------------------------------------------------------------------- |
| opensre    | [formula](./Formula/opensre.rb) | Open-source SRE agent for automated incident investigation and root cause analysis |

## Documentation

`brew help`, `man brew` or check Homebrew's documentation.
